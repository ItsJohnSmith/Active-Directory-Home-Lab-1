# Active Directory Home Lab (AWS Lightsail)

## Overview
Built a Windows Server Active Directory environment to simulate real-world helpdesk and system administration tasks.

## Key Features
- Deployed Windows Server as Domain Controller
- Created and managed users and groups
- Implemented Role-Based Access Control (RBAC)
- Configured shared folders (Sales, HR, IT)
- Set NTFS and share permissions
- Delegated helpdesk permissions for password resets
- Enforced password policy using Group Policy

## Simulated Helpdesk Tasks

- Reset user passwords in Active Directory
- Assigned users to department groups
- Configured file share access based on roles
- Troubleshot NTFS permission inheritance issues

## Structure
- Admins
- Employees
- HelpDesk
- Groups (Sales, HR, IT)

## File Shares
C:\Company Shares
- Sales (Sales group access)
- HR (HR group access)
- IT (IT group access)
